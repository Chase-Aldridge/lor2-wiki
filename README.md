## Overview

This wiki contains the **complete** content from three epic D&D campaigns set in the World of LOR:

- **[Aeternum - The Fellowship of the Blade](docs/aeternum.md)** - The Aeternum Crisis (1111 AE)
- **[Gul'Rath - The Continent of Intrigue](docs/gulrath.md)** - Post-war political intrigue
- **[Plagas - The Savage Frontier](docs/plagas.md)** - The Plagas Prophecy (1116 AE)

Plus comprehensive references for:
- **[Pantheon of the Gods](docs/gods.md)** - All deities, titans, and planar entities
- **[Game Mechanics](docs/mechanics.md)** - Bankai, Domain Expansion, power tiers

## Features

- **Wikidot-style design** - Clean, minimal, text-focused layout
- **Complete content** - Every character, faction, location, and story detail included
- **Dark mode default** - Easy on the eyes for long reading sessions
- **Local search** - Fast full-text search across all content
- **Mobile responsive** - Read on any device

## Local Development

### Installation

```bash
npm install
```

### Start Development Server

```bash
npm start
```

This starts a local development server at `http://localhost:3000/` with live reload.

### Build for Production

```bash
npm run build
```

Generates static content into the `build/` directory ready for deployment.

### Serve Production Build Locally

```bash
npm run serve
```

## Philosophy

Unlike typical wiki summaries, this project includes **everything** from the source campaign documents:

- No content summarization
- No detail omission
- Complete character, faction, and location information
- Full story arcs and campaign notes
- Comprehensive reference material

The goal is a complete, authoritative reference for players, DMs, and anyone interested in exploring the rich World of LOR in its entirety.

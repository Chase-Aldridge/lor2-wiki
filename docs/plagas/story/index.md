## Story Arcs (Planned)

The following story arcs have been outlined for the campaign:

1. **Arriving in Plagas** - Strangers in a wild land
2. **Pol Tarei: Swiftsail vs The Octopus** - Criminal underworld conflict
3. **The Contract** - Tragic mercenary expedition with the warforged
4. **Windrun and Return Home** - Awakening of powers, new heroes join up
5. **The Titan Hunt** - Major faction conflict
